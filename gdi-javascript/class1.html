More on variables

In programming languages variables have data types.

Define's what kind of values they can hold.

More important in some languages than others.

Data types

string: an immutable string of characters.


var greeting = 'Hello World.';
var restaurant = "Chipolte";

number: whole (6, -102) or floating point (5.8737).


var myAge = 25;
var pi = 3.14;

boolean: Represents logical values true or false.


var catsAreBest = false;
var dogsRule = true;

undefined: Represents a value that hasn't been defined.


var notDefinedYet;

null: Represents an explicitly empty value.


var goodJokes = null;

Functions

Re-usable chunks of code.

Need to be defined before they can be used.


function sayHello() {
  console.log('Hi Foti!');
}

Then call it all you want.


sayHello();

Beware of circular dependancies.


function chicken() {
    egg();
}

function egg() {
    chicken();
}

egg();

Arguments

Functions can accept arguments.

Extra pieces of information the function needs to do it's work.


function sayMyName(name) {
  console.log('Hi, ' + name);
}

sayMyName('Foti');
sayMyName('Testy McTesterFace');

  console.log(result);

Return values

Functions can also give things back.

Values that are returned as a result of the function.


function addNumbers(num1, num2) {
  var result = num1 + num2;
  return result; // Anything after this line won't be executed
}

var sum = addNumbers(5, 2);
console.log(sum);

Return values cont.

You can use function calls in expressions.


var biggerSum = addNumbers(2, 5) + addNumbers(3, 2);

You can even call functions inside function calls.


var hugeSum = addNumbers(addNumbers(5, 2), addNumbers(3, 7));

Scope

JavaScript variables have "function scope".

That means they are visible in the function where they're defined.

You can see scopes above you, but not below you.

A variable with "local" scope.


function addNumbers(num1, num2) {
  var localResult = num1 + num2;
  console.log("The local result is: " + localResult);
}
addNumbers(5, 7);
console.log(localResult);

A variable with "global" scope.


var globalResult;
function addNumbers(num1, num2) {
  globalResult = num1 + num2;
  console.log("The global result is: " + globalResult);
}
addNumbers(5, 7);
console.log(globalResult);

Use these operators to compare two values for equality, inequality, or difference.

These operatorse evaluate to true or false.

==Equal
!=Not Equal
<Less than

>=Greater than or Equal to

Logical Operators

&&And
||Or

When combining together multiple conditions, use parantheses to group.


var myAge = 28;
if ((myAge >= 0 && myAge < 3)  || myAge > 90) {
  console.log('You\'re not quite in your peak.');
}

Truthy vs Falsey

If you don't use a comparison or logical operator, JavaScript tries to figure out if the value is "truth-y".

Best to explicitly check for what you want.


var catsRule = true;
if (catsRule) {
  console.log('Yay cats!');
}

Values that are "false-y": false, the empty string (""), the number 0, undefined, null.


var name = '';
if (name) {
  console.log('Hello, ' + name);
}
var points = 0;
if (points) {
  console.log('You have ' + points + ' points');
}
var firstName;
if (firstName) {
  console.log('Your name is ' + firstName);
}

Short-Circuit Evaluation

JavaScript evaluates logical operators from left to right and stops evaluating as soon as it knows the answer.

(false && anything) => false

(true || anything) => true


var nominator = 5;
var denominator = 0;
if (denominator != 0 || (nominator/denominator > 0)) {
  console.log('Thats a valid, positive fraction');
}

The if/else statement

Use else to give an alternative statement to execute.

Must always used with an if.


var age = 28;
if (age > 16) {
  console.log('Yay, you can drive!');
} else {
  console.log('Sorry, but you have ' + (16 - age) + ' years til you can drive.');
}

The if/else if/else statement

How many times can I put if and else together.

As many as you like with the if/else if/else.

You can use else if if you have multiple exclusive conditions to check.


var age = 20;
if (age >= 35) {
  console.log('You can vote AND hold any place in government!');
} else if (age >= 25) {
  console.log('You can vote AND run for the Senate!');
} else if (age >= 18) {
  console.log('You can vote!');
} else {
  console.log('You have no voice in government!');
}

The while loop

Sometimes you want to do the same bit of code over again.

No reason to rewrite it. Just use a loop.


while (expression) {
  // statements to repeat
}

Loops continue to run, till the expression evaluates to false.


var x = 0;
while (x < 5) {
  console.log(x);
  x = x + 1;
}

What is an issue that could occur here.

The for loop

What if you had a better idea of how many times you want count to run.

Like a certain count.

This is where a for loop would come in handy.


for (initialize; condition; update) {
  // statements to repeat
}
					

for (var i = 0; i < 5; i = i + 1) {
  console.log(i);
}

Arrays

Can't talk about for loops without talking about arrays.

An array is a type of data-type that holds an ordered list of values, of any type.

Arrays and for loops are bff's.


var arrayName = [element0, element1, ...];
					

var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet'];
var raceWinners = [33, 72, 64];
var myFavoriteThings = ['Cake', 2113, 'Movies'];

The length property reports the size of the array.


console.log(rainbowColors.length);

You can access (or index) items with "bracket notation".

Take note: the index starts at 0.


var arrayItem = arrayName[indexNum];
					

var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet'];
var firstColor = rainbowColors[0];
var lastColor = rainbowColors[6];

You can also use bracket notation to change the item in an array.


var myFavoriteThings = ['Cake', 2113, 'Movies'];
myFavoriteThings[0] = 'Celery Root';

You can also use bracket notation to change the item in an array.


rainbowColors[4] = 'Black';

You can also use methods or functions.


rainbowColors.push('Silver');

Use for loops for easy iteration.

Easily process each item in an array.


var rainbowColors = ['Red', 'Orange', 'Yellow', 'Green', 'Blue', 'Indigo', 'Violet'];
for (var i = 0; i < rainbowColors.length; i++) {
  console.log(rainbowColors[i]);
}
